# Interactive Historical Perspectives Map

An interactive, multilayered web application with a timeline to visualize different perspectives on historical developments. This project allows users to explore historical data on a dynamic map, toggle different data layers, and filter the information based on a selected time range.

## Live Demo

**[View the live application here](https://historic-interactive-map.vercel.app/)**

---

## Technology Stack

- **Core Framework:** React with TypeScript
- **Build Tool:** Vite
- **Mapping Library:** React-Leaflet & Leaflet.js
- **UI Components:** Material-UI (MUI)
- **Data Format:** GeoJSON
- **Hosting & Deployment:** Vercel

---

## Running the Project Locally

To set up and run this project on your own machine, follow these steps:

### Prerequisites

- Node.js
- npm or yarn

### Setup Instructions

1.  **Clone the repository:**

    ```bash
    git clone https://github.com/bobzoob/historic-interactive-map.git
    ```

2.  **Navigate into the project directory:**

    ```bash
    cd historic-interactive-map
    ```

3.  **Install the dependencies:**

    ```bash
    npm install
    ```

4.  **Start the development server:**
    ```bash
    npm run dev
    ```

The application should now be running on `http://localhost:5173`.
